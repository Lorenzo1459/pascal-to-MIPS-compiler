INSTRUÇÃO PARA EXECUTAR:
(testado no ubuntu 20.04)
 - Faça download do arquivo .zip e extraia no seu computador.
 - Entre na pasta extraida, e no terminal digite: make
 - após dar make, é gerado o arquivo executável "trabcp1"
 - para rodar os testes, digite: ./trabcp1 < tests/XXXXXXXX.p 
 onde XXXXXXX é o nome do arquivo em pascal
