INSTRUÇÃO PARA EXECUTAR:
(testado no ubuntu 20.04)
 - Faça download do arquivo .zip e extraia no seu computador.
 - Entre na pasta extraida, e no terminal digite: make
 - após dar make, é gerado o arquivo executável "trabcp3"
 - para rodar os testes, digite: ./trabcp3 < tests/3XXXXXXX.p 
 onde 3XXXXXX é o nome do arquivo em pascal
 - Observação importante: Para o cp3, os arquivos em pascal que iniciam com '3' na frente, foram testados e funcionam conforme esperado no MARS.
